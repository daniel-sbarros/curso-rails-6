# Arquivos do Curso Rails 6 API

* Clonar o Projeto
git clone git@github.com:daniel-sbarros/curso-rails-6.git
cd curso-rails-6

* Instalar as dependencias do Ruby on Rails
bundle install

* Instalar as depedencias do Node.js
yarn install

* Criar os bancos de dados
rails db:create

* Criar as tabelas
rails db:migrate
